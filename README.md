# Test_case_Unity
Test case on Unity3d

Приложение состоит из 2х сцен – менеджера уровней и собственно сцены уровня. 

Менеджер уровней содержит в себе карту уровней. Для удобства дебага есть кнопки для сброса списка, 
а уровни несут в себе номер уровня и его состояние(открыт/закрыт/пройден)

Основные элементы программы (подгружаются при загрузке): сущность корабля со своим контроллером и контроллер уровня. 

Контроллер содержит в себе компонент PlayerView (взаимодействие с UI: обратный таймер – сколько времени 
нужно продержаться и количество жизней), главное управление кораблем и загруженную модель данных.

Класс Model маркирован [Serializable] и имеет методы загрузки и сохранения состояния, а также методы 
для работы с уровнями.

Контроллер уровня (он же Spawner) управляет пулами астероидов и пуль и спавнит астероиды.

Реализованы меню паузы, меню победы (по истечении таймера) и поражения (при окончании жизней).
